p🤖 AI Chatbot

An AI-powered chatbot built using Python, Flask, HTML, CSS, JavaScript, Ollama, and Gemma 3:1b.

🚀 Features

- ChatGPT-style interface
- AI responses using Ollama
- Friendly, Funny and Realistic response modes
- Asks the user's name before starting
- Responsive chat interface
- Runs completely locally

🛠️ Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Ollama
- Gemma 3:1b

▶️ How to Run

1. Install Python.
2. Install Ollama.
3. Run:
   ollama run gemma3:1b
4. Install dependencies:
   pip install flask requests
5. Start the application:
   py app.py
6. Open:
   http://127.0.0.1:5000
